# Course Feedback Icons

Put PNG files for the course evaluation buttons in these paths.

## Difficulty

- `difficulty/easy.png` - 쉬움
- `difficulty/normal.png` - 보통
- `difficulty/slightly-hard.png` - 약간 어려움
- `difficulty/hard.png` - 어려움
- `difficulty/extreme.png` - 매우 어려움

## Duration

- `duration/under-2h.png` - 2시간 이하
- `duration/2-3h.png` - 2~3시간
- `duration/3-4h.png` - 3~4시간
- `duration/4-5h.png` - 4~5시간
- `duration/over-5h.png` - 5시간 이상

The app will use these PNG files when present. If a file is missing, the UI falls back to the built-in icon so the page does not show a broken image.
